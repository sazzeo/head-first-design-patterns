# 데코레이터 패턴
- 데코레이터의 슈퍼클래스는 자신이 장식하고 있는 객체의 슈퍼클래스와 같다.
- 한 객체를 여러 개의 데코레이터로 감쌀 수 있다.
- 데코레이터는 자신이 장식하고 있는 객체에게 어떤 행동을 위임하는 일 말고도 추가 작업을 수행할 수 있다.

## 구성요소들
1. 추상 구성요소 : Beverage
2. 추상 데코레이터 : CondimentDecorator
3. 구상 구성요소 : Espresso , HouseBlend 등
4. 구상 데코레이터 : Whip , Mocha 등

## 주의점
- 구상 구성 요소로 어떤 작업을 처리하는 코드에 데코레이터 패턴을 적용하면 제대로 작동하지 않음
- 실제로는 팩토리나 빌더 패턴으로 데코레이터를 만듦.
- 데코레이터는 원래 클래스에 행동을 추가하는 용도로 사용되므로, 원래 클래스를 바꾸는 용도로 사용하는데는 적합하지 않음. 
- 데코레이터 패턴은 자잘한 객체가 매우 많이 추가될 수 있고, 남발하면 코드가 필요이상으로 복잡해짐.
